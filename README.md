# DevConnect Frontend

DevConnect is a developer networking and collaboration platform that enables users to connect, collaborate on projects, and grow their professional network.  
This repository contains the **frontend implementation** built using **React**, **Vite**, **Tailwind CSS**, **Redux Toolkit**, and **Axios**.

---

## ⚙️ Tech Stack
- **Frontend Framework:** React (Vite)
- **State Management:** Redux Toolkit
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios
- **Routing & UI:** React Router DOM
- **API Integration:** Connected with the DevConnect Backend (Node.js + Express.js)

---

## ✨ Features
- **Authentication:** Secure login and signup integrated with backend APIs.
- **User Feed:** Displays developer profiles with pagination.
- **Profile Management:** View and edit user profile details.
- **Connection Handling:** Send, accept, reject, or ignore connection requests.
- **Dynamic UI Components:** Reusable and scalable React components.
- **State Management:** Efficient global state handling with Redux Toolkit slices.
- **Responsive Design:** Optimized for all screen sizes using Tailwind CSS.
- **Error Handling:** Axios interceptors and Redux actions for clean API communication.
