# **DevConnect Frontend**

DevConnect is a developer networking and collaboration platform designed to help developers connect, communicate, and expand their professional network.  
This repository contains the frontend implementation built using **React, Vite, Tailwind CSS, Redux Toolkit, and Socket.io-client** for real-time communication.
